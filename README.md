# 💼 Portafolio Personal — Ramiro Usnayo

Sitio web de portafolio personal desarrollado con **HTML5** y **CSS3**, diseñado para presentar información profesional, proyectos, habilidades y medios de contacto de manera clara, moderna y responsive.

---

## 🚀 Características

- Diseño limpio, estructurado y fácil de navegar
- Navegación sencilla con menú
- Secciones bien organizadas
- Íconos de tecnologías con **Font Awesome**
- Formulario de contacto
- Links directos a correo y WhatsApp
- Preparado para diseño **responsive**

---

## 🧩 Secciones del sitio

| Sección | Descripción |
|---|---|
| 🧑 **Sobre mí** | Presentación personal e intereses en el desarrollo web |
| 💻 **Proyectos** | Galería con imagen, descripción y enlace a cada proyecto |
| 🛠 **Habilidades** | Tecnologías y capacidades en desarrollo web |
| 📬 **Contacto** | Formulario, correo electrónico y enlace a WhatsApp |
| 🌐 **Redes sociales** | Links a Instagram, LinkedIn, Facebook y X (Twitter) |
