💼 Portafolio Personal — Ramiro Usnayo
Este proyecto corresponde a un sitio web de portafolio personal desarrollado con HTML5 y CSS3, cuyo objetivo es presentar información profesional, proyectos desarrollados, habilidades y medios de contacto de manera clara, moderna y responsive.
El sitio está diseñado para ser simple, ordenado y fácil de navegar, destacando contenido relevante para reclutadores, clientes o colaboradores.
🚀 Características del proyecto
Diseño limpio y estructurado
Navegación sencilla con menú
Secciones organizadas
Íconos de tecnologías usando Font Awesome
Formulario de contacto
Links directos a correo y WhatsApp
Preparado para diseño responsive
